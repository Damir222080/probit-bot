# Бот-пробив

Бесплатный бот для поиска информации по номеру, имени, email и IP.

## Команды
- `/phone 79991234567`
- `/name Иван Петров`
- `/email test@mail.ru`
- `/ip 8.8.8.8`

## Деплой на Railway
1. Загрузи репозиторий на GitHub
2. Зайди на railway.app, подключи GitHub
3. Выбери репозиторий, добавь переменную BOT_TOKEN
4. Нажми Deploy
